DICOM Converter Python Script

> This script will convert any files under file extension .GZ to a folder of PNG Images (under "ImageExport")
> The files must be placed in the same directory as the main.py script
> Test Data used is from OpenNeuro. See:
	https://openneuro.org/
	https://openneuro.org/datasets/ds004471/versions/1.0.0

The Script was made using the following modules. Make sure they are installed before executing.
